# MetroLife UI/UX 設計規範 v1.0
**設計風格**: Modern Rounded / Soft UI / Glassmorphism  
**主色調**: 米白 (Warm Beige) / 深炭 (Dark Charcoal)  
**圓角系統**: 統一 12px-24px 柔和圓角  
**字體**: Inter (EN) / Noto Sans TC (HK)  

---

## 1. 設計系統 (Design System)

### 1.1 顏色規範
```css
/* Light Theme - 米白風格 */
--bg-primary: #FAF9F6;        /* 暖白色背景 */
--bg-secondary: #FFFFFF;       /* 卡片純白 */
--bg-tertiary: #F5F5DC;        /* 米白強調 (Beige) */
--accent-primary: #DE2910;     /* 香港紅 */
--accent-secondary: #FFD700;   /* 金色 */
--text-primary: #1C1C1E;       /* 深黑文字 */
--text-secondary: #6B6B6B;     /* 次要灰 */
--text-tertiary: #9E9E9E;      /*  placeholder */
--success: #34C759;            /* 綠色勾選 */
--warning: #FF9500;            /* 橙色警告 */
--danger: #FF3B30;             /* 紅色錯誤 */

/* Dark Theme - 深色護眼 */
--bg-primary-dark: #121212;    /* 深炭背景 */
--bg-secondary-dark: #1E1E1E;  /* 卡片深灰 */
--bg-tertiary-dark: #2C2C2E;   /* 懸浮背景 */
```

### 1.2 圓角系統 (Radius System)
| 元素類型 | 圓角值 | 用途 |
|---------|-------|------|
| **大卡片** | 24px | 主頁天氣卡、記帳圓形圖容器 |
| **中卡片** | 20px | 功能模組、設置選項 |
| **小卡片** | 16px | 列表項目、待辦事項 |
| **按鈕** | 14px | CTA 按鈕、導航按鈕 |
| **輸入框** | 12px | 文本輸入、搜索欄 |
| **圖標/頭像** | 50% (圓形) | 用戶頭像、勤力兔圖標 |
| **標籤** | 20px (膠囊形) | 分類標籤、狀態標籤 |

### 1.3 陰影規範 (Soft Shadows)
```css
/* 淺色模式陰影 */
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.04);
--shadow-md: 0 4px 16px rgba(0, 0, 0, 0.06);
--shadow-lg: 0 8px 24px rgba(0, 0, 0, 0.08);
--shadow-glow: 0 4px 20px rgba(222, 41, 16, 0.15); /* 紅色光暈 */

/* 深色模式陰影 */
--shadow-dark: 0 4px 16px rgba(0, 0, 0, 0.4);
```

### 1.4 間距系統 (8pt Grid)
- **xs**: 4px | **sm**: 8px | **md**: 16px | **lg**: 24px | **xl**: 32px | **xxl**: 48px

## 2. 全局組件庫 (Component Library)

### 2.1 底部導航欄 (Bottom Navigation)
```
高度: 80px (含安全區域)
背景: Glassmorphism (backdrop-filter: blur(20px))
圓角: 頂部 24px 圓角 (top-left, top-right)
陰影: shadow-lg

佈局:
┌─────────────────────────────────────────────┐
│  [📝]      [💰]      [🏠]      [🏃]      [⚙️] │
│  Daily   Finance   Home    Exercise   Settings│
│   (14px 圓角圖標, 選中時放大 1.1x + 紅色)      │
└─────────────────────────────────────────────┘
```
- **主頁按鈕**: 中央懸浮，56px 圓形，紅色背景，白色圖標，陰影 glow
- **選中狀態**: 圖標向上浮動 4px，底部顯示 4px 圓角紅色指示條

### 2.2 勤力兔組件 (Diligent Rabbit Widget)
```
尺寸: 120x120px (浮動)
位置: 右下角，距邊緣 24px
動畫: 呼吸效果 (scale 1.0 ↔ 1.05)
圓角: 50% (完美圓形)
陰影: 柔和投影

對話氣泡:
- 背景: 白色/深色卡片
- 圓角: 20px (左下角 4px 尖角指向兔子)
- 陰影: shadow-md
- 最大寬度: 200px
```

## 5. 響應式與適配 (Responsive & Adaptations)

### 5.1 無邊框設計 (Edge-to-Edge)
- **iPhone 動態島/Dynamic Island**: 頂部狀態欄文字使用黑色/白色 (依主題)
- **底部安全區**: 導航欄高度包含 34px (iPhone) 或 16px (Android) 安全邊距
- **水平邊距**: 全局 16px (手機) / 24px (平板)

### 5.2 深色模式適配 (Dark Mode)
| 元素 | 淺色模式 | 深色模式 |
|------|---------|---------|
| 背景 | `#FAF9F6` (暖白) | `#121212` (深炭) |
| 卡片 | `#FFFFFF` | `#1E1E1E` |
| 邊框 | `rgba(0,0,0,0.05)` | `rgba(255,255,255,0.1)` |
| 文字 | `#1C1C1E` | `#FFFFFF` (90% opacity) |
| 次要文字 | `#6B6B6B` | `rgba(255,255,255,0.6)` |

### 5.3 無障礙設計 (Accessibility)
- **最小觸控區域**: 44x44pt (iOS) / 48x48dp (Android)
- **對比度**: 文字與背景對比度 ≥ 4.5:1
- **字體縮放**: 支援系統字體大小 100%-200%
- **減少動效**: 支援 `prefers-reduced-motion` 設置

## 6. 動效規範 (Motion Design)

### 6.1 過渡動畫 (Transitions)
- **頁面切換**: 底部導航切換使用 `FadeThrough` (淡入淡出)，200ms
- **卡片展開**: 使用 `SharedAxis` 過渡，300ms，緩動 `easeInOutCubic`
- **底部面板**: 從底部滑入 (Slide up)，400ms，帶蒙層漸變

### 6.2 微交互 (Micro-interactions)
- **按鈕點擊**: Scale 0.95，100ms，反彈恢復
- **列表項滑動**: 右滑顯示操作 (Edit/Delete)，阻尼感
- **刷新**: 頂部下拉，圓形進度指示器旋轉
- **數字變化**: 金額/步數變化時，數字滾動動畫 ( odometer effect )

### 6.3 勤力兔動畫 (Diligent Rabbit Animations)
- **待機**: 輕微呼吸 (scale 1.0 ↔ 1.03)，3秒循環，無限
- **出現**: 從右下角彈入，旋轉 -10° → 0°，彈性緩動
- **說話**: 氣泡從兔子頭頂彈出，帶輕微搖擺
- **慶祝**: 跳躍 + 旋轉 360°，Confetti 爆炸效果
