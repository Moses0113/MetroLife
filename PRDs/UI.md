MetroLife UI/UX 設計規格文檔
1. 視覺語言與風格
設計哲學：現代簡約（Modern Minimalist），高留白感。

配色方案：

米白模式 (Light): 背景 #F5F5F2，主文字 #2D2D2D，強調色（勤力兔紫）#8E76FF。

黑色模式 (Dark): 背景 #1A1A1A，主文字 #E0E0E0，強調色 #BB86FC。

字體：系統預設（iOS: San Francisco, Android: Roboto）。

2. 推薦成熟 UI 組件 (Cursor 實施參考)
為確保穩定性與開發效率，建議 Cursor 優先使用以下封裝組件：

基礎 UI: flutter_riverpod (狀態), flex_color_scheme (主題管理)。

日曆: table_calendar (高度可定製的 Google Calendar 風格)。

圖表: fl_chart (處理記帳圓餅圖與運動棒形圖)。

動畫: lottie (用於「勤力兔」的動態反饋)。

交互: flutter_slidable (用於待辦事項與記帳的側滑刪除)。

玻璃擬態: glassmorphism (用於全屏番茄鐘介面)。

4. 特殊交互：全屏番茄鐘 (Landscape/Portrait)
佈局：大圓環計時器居中，字體粗大清晰。

動態效果：背景緩慢波動的「米白色/黑色」渐变。

鎖定功能：提供一個「鎖定」圖示防止誤觸退出。

勤力兔：在計時器下方露出半個頭，陪伴用戶專注。

5. 數據備份 UI (Setting 下屬)
位置：設定頁 -> 數據安全 -> 備份與還原。

按鈕 A：[ 📤 導出數據備份 (JSON) ] (調用文件選擇器儲存)。

按鈕 B：[ 📥 導入歷史數據 ] (顯示警告提示：導入將覆蓋現有本地數據)。
